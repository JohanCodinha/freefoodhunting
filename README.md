##Freefoodhunting
Free food hunting is a website to store tips about where yu can get free food.

As a user I can :
    -browser free food by location
    -add free food
    -comment on free food

A free food is a :
    -tip about a potential place where you could find free food
    propreties : 
            - location
            - photo
            - title
            - description
